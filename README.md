# KAH
### Kah é um script para busca automatizada por subdomínios.

## Requisitos
- Anew
- Assetfinder
- Findomain
- HTTPX
- Subfinder
- Takeover

## Como executar
```
make run
```
ou
```
make
```
ou
```
chmod +x ./src/*.sh && ./src/kah_sub.sh
```

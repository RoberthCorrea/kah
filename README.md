# KAH
### Kah é um script para busca automatizada por subdomínios.

## Requisitos
- Anew
- Assetfinder
- Findomain
- HTTPX
- Subfinder
- Takeover
- Nuclei
- Paramspider
- Kxss

## Para instalar as ferramentas
```
chmod +x ./ini.sh; ./ini.sh
```

## Como executar
```
make run
```
ou
```
make
```
ou
```
chmod +x ./src/*.sh && ./src/kah_sub.sh
```
